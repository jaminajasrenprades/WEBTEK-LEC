Introduction to Internet and Web Fundamentals

This course introduces the Internet, World Wide Web (WWW), World Wide Web Consortium (W3C), standard markup language and services of the internet.

Internet
Globally inforamation system that is logically linked together by a gloally uniquie address space based on the IP.
Vinton Cerf is the father of the Internet.
Network
It is used for connecting devices together.
Nodes
These are the devices that are connected.
Electronic Mail (E-mail)
First killer application in 1972.
World Wide Web
Way of accesing information over the medium of the internet. It is also an information-sharing model that is built on top of the internet.
Tim-Berners Lee is the Father of the World Wide Web.
Wide Area information Servers (WAIS)
The index of all databases.
Hypertext Transfer Protocol (HTTP)
Application layer communications protocol used to access resources (hypertext/hypermedia) on the World Wide Web.
Jointly developed by the World Wide Web Consortium(W3C) and Internet Engineering Task Force(IETF).
HTTP runs on top of TCP/IP, using TCP port 80 (TCP port 443 for HTTPS).

HTTP Message Header Fields

General Header Fields
  • Cache Control
    • This field is used to control the caching of a document, either a reponse or a request.
  • Connection
    • This field is used to control the connection and keep it alive.
  • Date
    • This field indicates the local time at which the message originated.
  • Pragma
    • This field specifies implementation-specific options.
  • Trailer
    • This field indicates the headers included in a checked trailer.
  • Transfer-Encoding
    • This field specifies the transfer encoding used in transferring an entity.
  • Upgrade
    • This field requests an upgrade or switch of protocols
  • Via
    • This field indicates the proxies used and also the proxy controls.
  • Warning
    • This field will give a warning if there is a possible problem

Entity Header Fields
  • Allow
    • This field shows what can you do or what can you allow with a specific header.
  • Content-Encoding
    • This field is used as a modifier of what type of media.
  • Content-Language
    • This field describes a page in a specific language.
  • Content-Length
    • Ths field shows you how many bites are there in your page or the entire body.
  • Content-Location
    • This field shows you where the content came from if the content is accessible from a location seperate from the requested resource's URI.
  • Content-MD5
    • This field digest the entity-body to show an end-to-end message integrity check of the entity body.
  • Content-Range
    • This field is used to range requests from byte 0-100.
  • Content-Type
    • This field shows you what type or kind of resource you use.(sample.txt/html)
  • Expires
    • This field gives the expiration date or time after the response is considered stale.
  • Last-Modified
    • This field shows you the last dated time of modification.
