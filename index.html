General Course Information

Course Description: Web Systems and Technologies

Course Code: 9427A

Class Hours: 1:00-2:00 TF

The course aims to provide an understanding of the basics of the Internet, how it originated, how it works, and how to develop and maintain Web sites.

Also, this website's purpose is to share the knowledge that we have learned throughout the semester.

Information Technology is a modern term that describes this combination of traditional computer and communication technologies.

Front-end Development

the practice of producing HTML, CSS and JavaScript.

World Wide Web

enabling the user to search for information.

Client-Side Scripting

the user's local computer connects to a server.

Server-Side Scripting

involves employing scripts on a web server.
