Glossary of Terms

This short glossary is intended to provide a useful and accessible set of definitions for the terms that we use in web systems and technologies.

ARPANET (Advanced Research Projects Agency Network)

- ARPANET was a network that was the basis for Internet. It was created for people to access computers, improve computer equipment, and to have a more effective communication method for the military much easier.

Adobe AIR (Adobe Integrated Runtime)

- Adobe AIR is a developer's tool, developed by Adobe, that can create platform-independent web applications that can be run on a user's desktop

AJAX (Asynchronous JavaScript and XML)

- AJAX is not a programming language or a tool, it is a client-side script that updates web pages asynchronously by exchanging data with a web server behind the scenes.

ANSI (American National Standards Institute)

- ANSI is the primary organization for fostering the development of technology standards in the United States.

ASP (Active Server Pages)

- A server-side script engine that enabled dynamically-generated web pages. ASP or Active Server Pages makes use of server-side scripting to dynamically produce web pages based on a specific request from the client.

BOM (Browser Object Model)

- BOM or Browser Object Model is used for all of the interactions between your code and the browser.

CFML (ColdFusion Markup Language)

- CFML is a web page markup language. It allows a website developer to create pages with variable information that is filled in dynamically in response to variables such as user input.

CGI (Common Gateway Interface)

- Transfering web user's request to a application program and to receive the data back to forward to the user.

DHTML (Dynamic HyperText Markup Language)

- DHTML refers to web contents that can change each time is viewed.

DNS (Domain Name System)

- DNS is a Internet Service that convert domain names to IP addresses.

DOM (Document Object Model)

- DOM is a application programming interface that is being developed by the W3C(World Wide Web). It gives a programmer or developer the ability to modify and create XML documents and HTML as full-fledged objects.

DTD (Document Type Definition)

- DTD is a set of markup declarations that defines the legal elements and the attribute and structure of an XML document.

EAR (Enterprise Archive)

- EAR is a JAR (Java Archive) file format that is used to package modules as a single archive file so that the deployment of the modules onto an application server happens consistently and simultaneously.

HTML (HyperText Markup Language)

- HTML is set of codes inserted in a file that is displayed on a World Wide Web browser page. It is used to create electronic documents called pages.

HTTP (HyperText Transfer Protocol)

- HTTP is the protocol that is used to move or transfer data over the World Wide Web.

HTTPS (Secure HyperText Transfer Protocol)

- HTTPS is an extension of HTTP that secures the exchange of file on the Wold Wide Web.
