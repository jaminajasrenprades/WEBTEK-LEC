    • Hypertext Markup Language (HTML)

    A markup language (not a programming language) that is used to create web pages in the World Wide Web.

    It was introduced by the rules of Standard Generalized Markup Language (SGML) in document type definition (DTD), by Tim-Berners-Lee in the year 1991.

    Creates structured documents by denoting semantics for text like headings, lists, links, quotes etc.

    HTML5 Elements

    header
    This element contains the header content of the site.
    footer
    This element contains the footer of the site.
    nav
    This element contains the navigation menu, or the other navigation functionality of the page.
    article
    This element is somewhat related to section tag. But it contains related individual standalone pieces of content. Ex: blog posts, videos, images, etc.
    section
    This element it used to group different articles.
    aside
    This element defines a block of content that is related to the main content around it, but not central to the flow of it.

    HTML VS. XHTML

    HTML and XHTML are quite related to each other but here are their possible differences.

    HTML XHTML
    Start or end tags are not required for every element. All elements must have a start tag.
    Tags and attributes are case-insensitive. Elements and attributes are case sensitive, it is mostly in lowercase.
    Attribues do not need to be quoted. Attribute values must be enclosed in quotes.
    The document must include an HTML5 DOCTYPE. XHTML documents must be well-formed.
    All the content can be put under the body element. All elements are nested properly.
