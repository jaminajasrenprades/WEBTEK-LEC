    HTML Elements

    Element is a part of a webpage, which includes an opening tag with some attributes, enclosed text content, and a closing tag.
    Head
    title
    base
    link
    meta
    This element is used to specify which character set is used, page description, keywords, author, and other metadata.
    style
    script
    This element defines the client-side JavaScripts.
    Body
    Grouping Elements span, div
    Headings h1, h2, h3, h4, h5 ,h6
    Paragraph and Break Lines p, br
    Link and Image Tags href, img, mailto
    Anchor tag - (a)
    List Tags ul, ol, dl, dt, dd, dir, menu

    HTML5 Elements

    header
    This element contains the header content of the site.
    footer
    This element contains the footer of the site.
    nav
    This element contains the navigation menu, or the other navigation functionality of the page.
    article
    This element is somewhat related to section tag. But it contains related individual standalone pieces of content. Ex: blog posts, videos, images, etc.
    section
    This element it used to group different articles.
    aside
    This element defines a block of content that is related to the main content around it, but not central to the flow of it.
